# 📌 Descripción del proyecto

**MoodMeter** es un sistema interactivo de visión por computadora capaz de analizar expresiones faciales humanas y estimar emociones mediante inteligencia artificial.

El proyecto utiliza la librería **DeepFace**, la cual implementa modelos de aprendizaje profundo basados en redes neuronales convolucionales (CNN) para detectar rostros y clasificar emociones.

El usuario puede cargar una imagen facial y el sistema devuelve una distribución probabilística de emociones:

- 😀 Happy
- 😢 Sad
- 😡 Angry
- 😨 Fear
- 😲 Surprise
- 😐 Neutral
- 🤢 Disgust

# 🎯 Objetivos de aprendizaje

Este proyecto permite:

- Comprender el papel de las redes neuronales en visión por computadora.
- Aplicar modelos de Deep Learning preentrenados.
- Utilizar la librería DeepFace para análisis facial.
- Crear una aplicación real basada en inteligencia artificial.
- Integrar un modelo de visión computacional con una interfaz web.
